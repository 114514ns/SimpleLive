# SimpleLive

重新构建的b站直播页面，简化的同时同时具有大多数日常需要用到的功能。

## 使用

复制[Tampermonkey.js](https://github.com/114514ns/SimpleLive/blob/master/Tampermonkey.js)的内容到油猴管理器

## 功能
- 匿名观看直播，不影响发弹幕和送礼物
- 多账号管理
